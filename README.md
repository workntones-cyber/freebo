# freebo

> **free**lance + **bo**okkeeping

フリーランスエンジニア向けのローカル動作会計アプリです。  
会計士に依頼せずに青色申告65万円控除を目指せるよう設計されています。

---

## 特徴

- 📒 **複式簿記対応** - 青色申告65万円控除に必要な複式簿記での記帳
- 🧾 **請求書管理** - 作成から入金済み処理まで一元管理
- 💱 **外貨対応** - 海外SaaSなど、USD建て経費の為替レート自動取得
- 💳 **クレカリマインド** - 引き落とし未処理の仕訳を自動検知・通知
- 📎 **領収書添付** - 仕訳に領収書ファイルを紐づけて保管
- 📈 **帳票出力** - 損益計算書・貸借対照表・総勘定元帳をPDF出力
- 🧭 **e-Tax転記ガイド** - 確定申告時の入力をサポート
- 🧮 **税額シミュレーション** - 年収に応じた所得税・住民税・手取りを試算
- 🖥️ **固定資産管理** - 減価償却の自動計算と仕訳生成
- 📅 **期限通知** - 確定申告・納税スケジュールをダッシュボードに表示
- 🌙 **ダーク/ライトモード** - 好みに合わせて切り替え可能
- 💾 **完全ローカル動作** - データはすべて手元のPCに保存

## 動作環境

- Windows 10 / 11
- Node.js v18以上
- npm v9以上

## セットアップ（エンジニア向け）

```bash
# リポジトリをクローン
git clone https://github.com/workntones-cyber/freebo.git
cd freebo

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev
```

### .exeファイルのビルド

```bash
npm run build:win
```

ビルド完了後、`dist\win-unpacked\freebo.exe` が生成されます。  
このフォルダごとコピーすれば別のPCでも動作します。

## 使い方の流れ

### 1. 初期設定

「設定」画面で以下を入力してください：

- 屋号・氏名・開業日
- 申告種別（青色65万円控除を推奨）
- 消費税モード（インボイス未登録の場合は「免税事業者」）
- 国民健康保険料・国民年金などの控除額（税額シミュレーション用）

### 2. 日々の入力

**経費を払ったとき（例：SaaSをクレカで購入）**

「仕訳入力」から：
```
日付：2026/04/21
摘要：Cloudflare 月額契約
支払方法：クレジットカード
借方：消耗品費　1,830円
貸方：未払金　　1,830円
```

クレカ払いの場合、仕訳帳に ⚠️ マークが付きます。  
翌月の引き落とし時に「引き落とし処理」ボタンで決済仕訳を自動生成できます。

**売上が発生したとき**

「請求書」から請求書を作成 → 「仕訳を自動生成」で売掛金の仕訳が自動登録されます。  
入金確認後は「入金済みにする」ボタンを押すだけです。

### 3. 確定申告時

1. 「帳票」画面で損益計算書・貸借対照表を確認
2. 「e-Taxガイド」画面でe-Taxへの転記内容を確認
3. 国税庁のe-Taxサイトで申告書を作成・送信

## 対象ユーザー

- 個人事業主（フリーランスエンジニア）
- 免税事業者（売上1,000万円以下・インボイス未登録）
- 青色申告65万円控除を受けたい方
- 源泉徴収なし（全額受取）の契約が中心の方

> ⚠️ 課税事業者・インボイス登録済みの場合は設定から切り替えできますが、消費税の仕訳は手動対応が必要です。

## データの保存場所

| 種類 | 保存場所 |
|---|---|
| 仕訳・請求書データ | `%APPDATA%\freebo\freebo.db`（SQLite） |
| 領収書ファイル | `プロジェクトフォルダ\exports\receipts\` |
| PDF出力 | `プロジェクトフォルダ\exports\年度\` |

> ⚠️ `exports/` フォルダは `.gitignore` で除外されています。領収書・PDFは別途バックアップしてください。
