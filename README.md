# freebo

> **free**lance + **bo**okkeeping

フリーランスエンジニア向けのローカル動作会計アプリです。
会計士に依頼せずに青色申告65万円控除を目指せるよう設計されています。

---

## 特徴

- 📒 **複式簿記対応** - 青色申告65万円控除に必要な複式簿記での記帳
- 🟢 **簡単入力モード** - 借方・貸方を意識せず「支払った／受け取った」で記録
- ⚙️ **詳細入力モード** - 複合仕訳など自由に入力できる上級者向けモード
- 🧾 **請求書管理** - 作成→PDF送付→入金済みをステータスバーで管理
- 💱 **外貨対応** - USD建て経費の為替レート自動取得（ECB→七十七銀行のフォールバック）
- 📎 **領収書・請求書添付** - 仕訳に証拠書類ファイルを紐づけて保管
- 🗂️ **領収書管理画面** - 添付した領収書を一覧・検索・フィルター
- 💳 **クレカリマインド** - 引き落とし未処理の仕訳を自動検知・通知
- 📈 **帳票出力** - 損益計算書・貸借対照表・総勘定元帳をPDF出力
- 🧭 **e-Tax転記ガイド** - 確定申告時の入力をサポート
- 🧮 **税額シミュレーション** - 年収に応じた所得税・住民税・手取りを試算
- 🖥️ **固定資産管理** - 減価償却の自動計算と仕訳生成
- 📅 **期限通知** - 確定申告・納税スケジュールをダッシュボードに表示
- 🌙 **ダーク/ライトモード** - 好みに合わせて切り替え可能
- 💾 **完全ローカル動作** - データはすべて手元のPCに保存

## 動作環境

- Windows 10 / 11（動作確認済み）
- macOS・Linux（ソースからビルド可能）
- Node.js v18以上
- npm v9以上

## セットアップ（エンジニア向け）

```bash
# リポジトリをクローン
git clone https://github.com/workntones-cyber/freebo.git
cd freebo

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev
```

### .exeファイルのビルド

```bash
npm run build:win
```

ビルド完了後、`dist\win-unpacked\freebo.exe` が生成されます。
このフォルダごとコピーすれば別のPCでも動作します。

## 使い方の流れ

### 1. 初期設定

「設定」画面で以下を入力してください：

- 屋号・氏名・郵便番号・住所・電話番号・メールアドレス
- 振込先（銀行名・支店名・口座種別・番号・名義）→ 請求書PDFに自動反映
- 申告種別（青色65万円控除を推奨）
- 消費税モード（インボイス未登録の場合は「免税事業者」）
- 国民健康保険料・国民年金などの控除額（税額シミュレーション用）

### 2. 日々の入力

**簡単入力モード（初心者向け）**

「仕訳入力」→「🟢 簡単入力」タブから：

```
💸 支払った（経費）
  何に使った：SaaS・ソフトウェア
  どうやって払った：💳 クレジットカード
  いくら：1,830 円
  いつ：2026/04/21
  メモ：CloudFlare ドメイン取得

→ 借方：通信費 / 貸方：未払金 が自動生成
```

```
💰 受け取った（収入）
  何の収入：開発・制作（請負）
  どうやって受け取った：🏦 銀行振込
  いくら：100,000 円
  いつ：2026/05/01

→ 借方：普通預金 / 貸方：売上高 が自動生成
```

**クレカ払いの場合**

仕訳帳に ⚠️ マークが付きます。クレカ明細が届いたら「引き落とし処理」ボタンで決済仕訳を自動生成できます。為替差損益も自動計算されます。

**外貨（USD）払いの場合**

USDを選択すると、ECB（欧州中央銀行）参考レートを自動取得して円換算します。レートの出典・日付が摘要に自動記録されます。

### 3. 請求書管理

「請求書」画面でステータスバーに沿って管理：

```
① 作成済み → ② PDFを作成して送付する → ③ 入金済みにする
```

- ② を押すとPDFが生成され、自動で「送付済み」ステータスになります
- 既存の請求書がある場合は「簡単入力」→「受け取った」から直接仕訳を登録できます

### 4. 確定申告時

1. 「帳票」画面で損益計算書・貸借対照表を確認・PDF出力
2. 「e-Taxガイド」画面でe-Taxへの転記内容を確認
3. 国税庁のe-Taxサイトで申告書を作成・送信

## 対象ユーザー

- 個人事業主（フリーランスエンジニア）
- 免税事業者（売上1,000万円以下・インボイス未登録）
- 青色申告65万円控除を受けたい方
- 給与所得との併用も想定（給与所得はfreeboでは管理不要）

> ⚠️ 課税事業者・インボイス登録済みの場合は設定から切り替えできますが、消費税の仕訳は手動対応が必要です。

## データの保存場所

| 種類 | 保存場所 |
|---|---|
| 仕訳・請求書データ | `%APPDATA%\freebo\freebo.db`（SQLite） |
| 領収書ファイル | `%APPDATA%\freebo\exports\receipts\年度\` |
| 請求書PDF | `%APPDATA%\freebo\exports\年度\invoices\` |
| 帳票PDF | `%APPDATA%\freebo\exports\年度\` |

> ⚠️ `%APPDATA%\freebo\` フォルダを定期的にバックアップしてください。

---

## 証拠書類の保管について

| 書類 | freeboでの管理 | 保管期間 |
|---|---|---|
| 領収書・レシート | 仕訳に添付（領収書管理画面で一覧確認） | 7年 |
| 請求書（発行） | 請求書画面でPDF管理 | 7年 |
| 総勘定元帳・仕訳帳 | DBのまま保管（PDF出力も可） | 7年 |
| 損益計算書・貸借対照表 | PDF出力して保管 | 7年 |

税務調査があった場合は、freeboを開いて仕訳帳・領収書を提示できます。

---

## 為替レートについて

外貨建て取引の円換算は以下の順でレートを自動取得します：

1. **ECB（欧州中央銀行）参考レート** - 当日のレートを取得
2. **七十七銀行 TTM** - ECBで取得できない場合（最大7日遡る）
3. **手動入力** - 両方失敗した場合

取得したレートの出典・日付は摘要に自動記録されます。継続して同じソースを使用することで、税務調査時の説明が容易になります。
